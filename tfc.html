The Four Collection

Freelance Work

2018 - Rebranding

Description:

The Four Collection is a Christian unisex clothing brand. As their designated Web Designer, I designed and created their first website in 2018, which launched in 2019. The aim was to create a simple e-commerce website that meets their needs and was easy for the end user to use. I am currently updating and rebranding their site.

Software Used:

The software I use is mainly Adobe Illustrator and I code with HTML, CSS & Javascript.

Web Design & Development

As the company is going through a rebrand, I am currently updating the website in alignment with brand style and to maintain brand consistency. Please see a video of the previous version of the website.
