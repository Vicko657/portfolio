The Four Collection

Freelance Work

2018 - Rebranding

Description:

The Four Collection is a Christian unisex clothing brand. As their designated Web Designer I was given the role to design and create their first webiste in 2018 and it launched in 2019. The aim was to create a simple eccommerce website that meets their needsand was easy for the end user to use. Currently I am updating and rebranding their site.

Software Used:

The software I use is mainly Adobe Illustrator and I code with HTML, CSS & Javascript.
