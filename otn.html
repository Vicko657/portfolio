Web Design & Development

2019

Description:

On The Narrow is Christian platform that aims to help and encourage other Christians with their Faith. They needed an e-commerce website to help showcase and sell their clothing apparel. This was a draft mock-up for the website.

Software Used:

The software I use is Adobe Illustrator for the design & Wordpress for the developement.

Web Design & Development

I designed and created the 'On The Narrow' website and ensured that it remained consistent with gold, black and white colour scheme. This was done to make sure that the end user would become familiar with the branding and recognise the brands products.
